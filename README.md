# ***"The Odin Project":*** Etch-a-Sketch Project

In this repository, you can find a webpage which contain a browser version of something between a **Sketchpad** and an **Etch-a-Sketch**. To use it, set the canvas size and hover the cursor over the grid to color the squares; the more you hover over them, the less opaque they become.

This project is build using HTML, CSS and JavaScript, employing Flexbox and DOM manipulation to make the whole grid system.
